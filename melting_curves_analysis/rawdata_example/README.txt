#melting_example.txt
The melting details (fluorescence ~ temperature for each well). To obtain this file in the Roche480 software, you go into melting curve analysis and right-click on the melting curve figures => export data => .txt


#ampli_example.txt
The amplification details (fluorescence ~ cycle for each well). To obtain this file in the Roche480 software, you right-click on the amplification phase figure => export data => .txt
